Spartan Mini

Expansion Board with Power

This project template is the basis of an expansion board for the Spartan Mini FPGA development board.

This base project includes a PCB edge defined as the same size as the Spartan Mini PCB with the connectors placed correctly to align the two boards. All IO present on the Spartan Mini board is connected to the project through the 0.1" expansion headers.
